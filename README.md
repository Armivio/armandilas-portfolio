# armandilas.com

Personal portfolio site. Built with [Astro](https://astro.build), deployed on [Vercel](https://vercel.com).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to GitHub → Vercel auto-deploys from `main` branch.

Configure custom domain `armandilas.com` in Vercel dashboard → Settings → Domains.
